# Programming Project OTP

# Author: Zack Chand

## Description ##

This program uses a Time - Based One Time Password (TOTP) algorithm to generate One Time Passwords. 

## Installation ##

These are the dependencies that needs to be installed before execution:

Navigate to program directory

1  pip install sego  
2. pip install Pillow 
3. Pip install pyotp

## Execution ##

To run program:

1 . Get OTP

python3 submission.py --get-otp

2. Generate QR

python3 submission.py --generate-qr
